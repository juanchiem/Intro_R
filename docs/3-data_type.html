3  Tipos de datos

En t√©rminos gen√©ricos, todos los elementos que maneja R se consideran objetos: un valor num√©rico, un vector, una funci√≥n, una base de datos, un gr√°fico, todos son objetos.

Uno de los t√≠picos errores que se nos presenta al usar R es el mal uso de las distintas clases de objetos que manipulamos en nuestras sesiones de trabajo.

Para un procesamiento correcto, un lenguaje de programaci√≥n debe saber qu√© se puede y qu√© no se puede hacer con un valor en particular. Por ejemplo, no se pueden hacer operaciones matem√°ticas con las palabras ‚Äúhola‚Äù y ‚Äúmundo‚Äù. Del mismo modo, no puede cambiar los n√∫meros 1 y -34.5 de min√∫sculas a may√∫sculas. 3.2 Factores

En R las variables categ√≥ricas se almacenan como factores tanto sea para vectores que contienen caracteres o num√©ros. Uno de los usos m√°s importantes de los factores es en el modelado estad√≠stico, asumiendo un rol de variable categorica, diferente de las variables cont√≠nuas. 3.4 N√∫meros aleatorios

La generaci√≥n de n√∫meros aleatorios es en muchas ocasiones un requerimiento esencial en investigaci√≥n cient√≠fica. Proceder de este modo puede reducir cualquier sesgo generado por nuestra persona a la hora de seleccionar una muestra, o aplicar un tratamiento a una unidad experimental.

  • Generar n√∫meros aleatorios de una distribuci√≥n espec√≠fica de par√°metros conocidos:
